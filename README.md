# AirHealthLink

AirHealthLink is a project dedicated to studying the various health effects of air pollution (PM2.5). 

---


## Scope

Data is taken from all available counties on the [AQS API](https://aqs.epa.gov/aqsweb/documents/data_api.html) from January 1, 2022 to December 31, 2022. 

---


## Usage/Replication

1. Create a local clone of the repository on your machine.
2. [Sign up for an AQS API key](https://aqs.epa.gov/aqsweb/documents/data_api.html#signup) and put it in [src/private_keys.py](https://github.com/leptio/AirHealthLink/blob/main/src/private_keys.py).
3. Run /src/main.py.
